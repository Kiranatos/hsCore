Terms:

JavaScript — язык программирования сценариев(script) со слабой типизацией и динамическим приведением типов.

Язык сценариев — язык системной интеграции. Языки, которые работают с готовыми программными компонентами.

Приклади движків JavaScript: MS Internet Explorer 11, MS Edge, Gecko (Moz;//a), Webkit (Safari), Blink (Chrome, Chromium, Opera, Yandex...)

DOM — Document Object Model. Работа со страницей: разметка, события

BOM — Browser Object Model. работа с браузером: история, хранилища, окно, cookies, таймеры, таймауты

Программирование = Алгоритмы + Структуры данных

Варінти підключення JavaScript:

- JavaScript-код в блоці body: Example 1001

- JavaScript-файл в блоці head: Example 1002. Але так як html-документ виконується порядково, то спочатку завантажиться і виконається JavaScript-файл. І тільки потім продовжиться прорисовка наступних тегів документа. Більш того: якщо скриптам буде потрібен якись html-об'єкт нижче, який ще не намальований, - то виникне помилка.

- Тому JavaScript-файл можна підключити в кінці блоку body: Example 1003. Щоб спочатку загрузився весь документ, а вже потім виконався сам скріпт.

Чтобы не подключать в конце html-файла, в файле javascript нужно добавить метод-шапку(для ожидания формирования DOM-дерева) из библиотеки JQuery - которая позволяет дожидаться загрузки документа:

$(document).ready(function(){

...........

});

Её сокращённый вариант:

$(function(){

...........

});
